E&K CLEANING SERVICE — One-Page Site
--------------------------------------
Files:
 - index.html        -> customized landing page
 - styles.css        -> dark theme styling
 - images/logo.png   -> your logo (copied from your upload)

Editing tips:
 - Replace text in index.html as needed (service areas, guarantee, etc).
 - Drop your own photos into images/ and add <img> tags where you want.
 - The "Get a Free Quote" form opens an SMS to 959-208-2935. To use email instead,
   replace the window.location line in handleQuote() with a mailto: link.

Free Hosting Options:
 - GitHub Pages: host the static files for free.
 - Netlify / Vercel: drag-and-drop deploy, free tier.

Publishing (GitHub Pages):
 1) Create a new public repo, upload all files.
 2) Settings → Pages → Build from main branch / root.
 3) Your site will go live at https://<username>.github.io/<repo>.
